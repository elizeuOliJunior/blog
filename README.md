# Projeto Meu Blog Pessoal

Este projeto é uma aplicação de blog pessoal, dividido em frontend e backend. Ele permite aos usuários criar, editar e excluir postagens, além de adicionar comentários.

## Como Funciona

- **Frontend**: Desenvolvido em React, o frontend oferece uma interface interativa para os usuários interagirem com o blog. Ele consome uma API fornecida pelo backend para exibir e gerenciar postagens e comentários.

- **Backend**: Desenvolvido em Flask, o backend oferece uma API que o frontend utiliza para realizar operações CRUD no banco de dados. Ele gerencia postagens e comentários, garantindo uma interação segura e eficiente.

## Configuração do Projeto

- Consulte os README específicos em `frontend/` e `backend/` para obter informações detalhadas sobre a configuração e estrutura de cada parte do projeto.
